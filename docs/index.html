略歴

金本 貴志（かねもと たかし）と申します。Webエンジニア/ITコンサルタントです。1984年2月18日生まれです。
静岡大学大学院情報学研究科修士課程を修了後、2008年に新卒で富士通株式会社に就職しました。
携帯電話端末のソフトウェアを開発するエンジニアとして、2011年末までの約4年間在籍しました。

その後、2011年設立の株式会社カルテットコミュニケーションズ（以下「カルテット」）という名古屋の広告代理店ベンチャーに、2012年1月より取締役として入社しました。
2020年3月末までの約8年間在籍し、ゼロから自社サービスとその開発組織を立ち上げるなど、CTOとして幅広い責務を担当し、在籍した8年間で売上規模5000万円→27億円、従業員数2名→60名という急成長に経営者として貢献しました。

2020年4月より独立してフリーランスエンジニアになりました。
引き続き名古屋を拠点とし、Webシステムの受託開発と社外CTO/技術顧問といった形での技術経営コンサルティングを主軸に活動してきました。
2022年4月より株式会社Kannade（カンナデ）を設立し、フリーランスエンジニアとして運営していた事業を法人として引き続き運営しています。
